# Short Link App

## Project Description
An application where user can shorten any link and also track the number of visits to the actual link using the short link

## Technology Used
* MeteorJS, NodeJS, Mongodb - (Back-end)
* ReactJS, SCSS, HTML - (Front-end)

## Project Setup
* Clone the repository on your system .
* Run the command npm install to install all node modules.
* Run command 'meteor' to start the application .
